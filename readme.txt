Контроллер STM32F103C8T6

PA0 (GPIOA0) - минус звукоизлучателя
PA3 (GPIOA3) - плюс звукоизлучателя
PA4 (GPIOA4) - кнопка относительно минуса
PA5 (GPIOA5) - зелёный светодиод относительно минуса
PA6 (GPIOA6) - красный светодиод относительно минуса
PA7 (GPIOA7) - вход АЦП контроль напряжения

Среда разработки:
STM32CubeMX 6.8.0
Keil uVision5 5.38
